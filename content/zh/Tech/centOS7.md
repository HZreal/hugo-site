# centOS7

安装yum
查看端口 netstat -anp |grep 80

安装wget
yum install wget

systemctl的所有service配置文件目录为/usr/lib/systemd/system

watch 命令监听
watch -d -n 2 监听执行某个命令，每2秒刷新一次

top 命令实时查看CPU内存

// 查看内存，每2秒刷新一次
free -m -s 2
